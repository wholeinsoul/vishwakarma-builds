Tech Stack Summary

LayerTechnologyWhy
FrameworkNext.js 14 (App Router)SSG for SEO, client components for wizard, Vercel deploy
DatabaseSupabase (PostgreSQL)Free tier, auth, RLS, real-time for RSVP updates
AIOpenAI GPT-4o-miniFast, cheap (~$0.02/plan), structured JSON output
StylingTailwind + shadcn/uiPolished components, print-friendly with @media print
PaymentsStripe CheckoutPer-party $4.99, no subscription complexity
EmailResendRSVP notifications, free tier
Rate LimitingUpstash RedisServerless-friendly, RSVP spam protection
HostingVercelFree tier, edge caching, instant rollback
Design Philosophy: Planning + RSVP in one tool is the wedge. The plan generator is table stakes (PartyGeniusAI proved it). Our differentiator: the RSVP tracker integrated into the party plan, themed templates with genuinely delightful output, and a print-friendly format parents actually use day-of.
