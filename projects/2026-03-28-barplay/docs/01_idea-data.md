# IdeaBrowser Idea of the Day — March 28, 2026

## Title
**QR Code Icebreaker Game for Awkward Social Events** (Barplay)

## URL Slug
`conversation-starter-engine-for-bars`

## Problem Description
Bars sell atmosphere. The promise is social energy, new people, a night worth having. The reality for anyone not arriving in a group is a phone in hand and a drink to nurse. Bar owners spend money on lighting, music, and decor to create the conditions for connection. Nothing on the table does anything once people sit down.

## Solution
Barplay puts a QR code on every table. Scan it and a trivia question appears. Answer it and a message pops from someone two tables over who got the same question wrong. Works in the browser. No download, no account. Bar owners subscribe for game packages and analytics. Every session captures data a POS system never touches: which formats extend dwell time, which nights drive cross-table interaction, which configurations bring people back.

## MVP Strategy
Start with 5 bars and 3 game formats that work in loud, distracted environments. Build the QR generator and basic WebSocket chat first. The success metric isn't scan rate — it's whether two strangers are still talking when the second round arrives. Run the first installs in person: show up, watch what gets ignored, rebuild what doesn't land.

## Revenue Model
- $49-$99/month per location
- Dwell time data = renewal argument
- Hospitality chains = institutional buyer
- Drinks brands sponsor themed game nights (media layer on subscriptions)
