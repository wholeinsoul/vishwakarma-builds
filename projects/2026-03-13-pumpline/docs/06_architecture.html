Tech Stack Summary

LayerTechnologyWhy
FrameworkNext.js 14 (App Router)SSG/ISR for SEO, React for interactivity
DatabaseSupabase (PostgreSQL)Free tier, RLS, built-in auth for admin
ORMPrismaType-safe queries, clean migrations
StylingTailwind CSSFast to build, responsive
HostingVercelFree tier, edge caching, one-click deploy
EmailResendFree tier, simple API
Rate LimitingUpstash RedisServerless-friendly, free tier
ValidationZodTypeScript-first schema validation
Design Philosophy: Static-first architecture. All public pages are pre-rendered at build time (ISR). Only API routes (review submission, lead capture, search) hit the database at request time. This means near-zero latency for SEO-critical pages and minimal database load.
